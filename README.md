# Conference Program App

A lightweight, reusable web application for displaying multi-day, multi-venue conference schedules. This app parses a Google Sheets CSV export into a structured JSON file, rendering a responsive schedule with "Favorites" functionality for attendee personalization.

## üöÄ Features

* **Dynamic Scheduling:** Automatically groups events by Day, Venue, and Time based on a flat CSV structure.
* **Google Sheets Integration:** Uses a Google Sheet as the CMS. A simple PHP script fetches and converts the live CSV to a static JSON file.
* **Multi-Day Tabs:** Automatically detects unique days from the data and generates navigation tabs.
* **Favorites System:** LocalStorage-based "Favorites" list allows users to star sessions and view a personalized schedule without creating an account.
* **Responsive Design:** Optimized for mobile and desktop viewing.
* **Modal Views:** Detailed session information (Abstracts, bios, authors) displays in a modal overlay.
* **Highlights Tab:** Dedicated view for keynote or highlighted sessions.
* **Flexible Layout:**
    * **Vertical:** Venues are stacked sequentially.
    * **Horizontal:** Concurrent sessions within a specific venue block are aligned side-by-side using Flexbox.

## üõ†Ô∏è Setup & Installation

### 1. Data Source (Google Sheets)

1. Create a Google Sheet with the following headers (case-sensitive as per `program-update.php`):
* **Venue Info:** `Day`, `Venue Date`, `Venue Hour`, `Venue Duration`, `Venue Location`, `Venue Title`, `Venue Room`, `Venue Moderation`
* **Session Info:** `ID` (Unique), `Highlight` (0 or 1), `Session Type`, `Session Duration` (pct), `Session Thumbnail`, `Session Title`, `Session Author(s)`
* **Details:** `Affiliations`, `Abstract`, `References`, `Keywords`, `Biographies`


2. Publish the sheet to the web: `File > Share > Publish to web > CSV`.
3. Copy the generated CSV link.

### 2. Backend Configuration

1. Open `program-update.php`.
2. Replace `$csvUrl` with your Google Sheets CSV link:
```php
$csvUrl = 'YOUR_GOOGLE_SHEETS_CSV_URL';

```


3. Upload the files to a PHP-enabled server.

### 3. Usage

* **Update Data:** Trigger the update manually by visiting `https://your-domain.com/program-update.php`. This will refresh `program.json`.
* **View Program:** Navigate to `index.html` to view the conference schedule.
